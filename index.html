Генератор задач по информатике

Типы заданий

  • Бит чётности
  • Калькулятор
  • Динамика

Добро пожаловать!

Данный генератор позволяет автоматизировать процесс составления задач по информатике для учеников 10-11 классов. В него включены 3 типа заданий (указаны слева) с 3 уровнями сложностями (A, B, C). Чтобы воспользоваться им, нажмите на нужное задание слева от данной инструкции. Для запуска генерации преподавателю достаточно в специально отведённом поле ввести 5-значный цифровой код. В результате появится уникальное задание, соответствующее выбранной тематике. Сформированное упражнение будет всегда доступно по тому же идентификатору. Ответы к заданию аккуратно спрятаны под условием.
