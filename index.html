Генератор задач по информатике

Типы заданий

  • Бит чётности
  • Калькулятор
  • Динамика

Добро пожаловать!

Данный генератор позволяет автоматизировать процесс составления задач по информатике для учеников 10-11 классов.
В него включены 3 типа заданий с 3 уровнями сложностями (A, B, C). Чтобы воспользоваться генератором, нажмите на нужное задание слева от данной инструкции.

Для запуска генерации преподавателю достаточно в специально отведённых полях ввести уникальный цифровой код (рекомендуемая длина – 5 цифр) и количество заданий (различных вариантов). В результате появится набор заданий, соответствующий выбранной тематике. Сформированные упражнения будут всегда доступно по тому же идентификатору.

Ответы к заданиям аккуратно спрятаны под условием.
